# Update Docs Command

「単一の真実のソース（Single Source of Truth）」からドキュメントを同期します。

1. **`package.json` の scripts セクションの読み取り**
- scripts 参照テーブルの生成
- コメント内の説明文の包含


2. **`.env.example` の読み取り**
- すべての環境変数の抽出
- 用途と形式の記録


3. **`docs/CONTRIBUTING.md` の生成**（以下を含む）：
- 開発ワークフロー
- 利用可能な scripts
- 環境設定
- テスト手順


4. **`docs/RUNBOOK.md` の生成**（以下を含む）：
- デプロイ手順
- 監視とアラート
- よくある問題と修正方法
- ロールバック手順


5. **古くなったドキュメントの特定**：
- 90日以上更新されていないドキュメントの特定
- 手動レビュー用のリスト作成


6. **差異サマリーの表示**

**単一の真実のソース：** `package.json` および `.env.example`
