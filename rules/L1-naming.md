# Naming Rules

## Directory Names

- kebab-case

## File Naming

- **File Name**: kebab-case
- **TypeScript Type Files**: kebab-case with .types suffix. e.g., `market.types.ts`

## Variables and Functions

- **Variables and Functions**: camelCase
- **Components, classes, interfaces, enums, and type aliases**: PascalCase
- **Constants**: UPPER_SNAKE_CASE
