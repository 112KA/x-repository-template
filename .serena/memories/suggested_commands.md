# Suggested commands (Linux)
## Setup
- pnpm install

## Dev servers
- pnpm dev:nextjs
- pnpm dev:astro
- pnpm dev:vite

## Watch/build packages
- pnpm watch
- pnpm watch:x
- pnpm watch:x3
- pnpm build
- pnpm build:x
- pnpm build:x3

## Lint/format
- pnpm lint
- pnpm lint:fix

## Tests
- pnpm testAll
- pnpm x test

## App-specific shortcuts
- pnpm app:nextjs <command>
- pnpm app:astro <command>
- pnpm app:vite <command>
- pnpm x <command>
- pnpm x3 <command>
- pnpm x-lib <command>

## Next.js page definition tooling
- pnpm app:nextjs generate:pages

## System utilities (Linux)
- ls, cd, pwd
- cat, less
- grep, rg
- find
- git status, git diff, git log
